# AI Customer Support Chatbot (NLP)

## Overview
This project is an AI-powered customer support chatbot developed using Python and Natural Language Processing (NLP).

The chatbot can understand user queries, classify the intent using machine learning, and provide appropriate responses.

A simple interactive interface is created using Gradio.

---

## Features

- NLP based chatbot
- Intent classification using Machine Learning
- Interactive chatbot UI using Gradio
- Implemented using Python and Scikit-learn

---

## Technologies Used

- Python
- Natural Language Processing (NLP)
- Scikit-learn
- Gradio
- Google Colab

---

## How It Works

1. User enters a query in the chatbot interface.
2. The text is converted into numerical features using CountVectorizer.
3. A Logistic Regression model predicts the intent of the query.
4. The chatbot returns a predefined response based on the predicted intent.
